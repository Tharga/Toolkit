# Tharga Toolkit Standard

General tools with extensions for collections, DateTime, Luhn and Compare.

### Features
- Compare
- ManagedTimer
- DateTimeExtensions
- ListExtensions
- HashExtensions
- EnumExtensions
- Enumeration
- Luhn
